## Project Overview
This project is a **full-stack healthcare web application** designed to simplify and digitize on-campus medical services for students. The platform enables students to **book doctor consultations, request emergency ambulance services, and securely access digital prescriptions** through a single unified interface.

The system reduces dependency on manual coordination and improves accessibility, response time, and reliability of student healthcare services.

---

## Core Features
- Online booking for doctor consultations  
- Emergency ambulance request functionality  
- Secure access to digital prescriptions and medical records  
- Centralized healthcare dashboard for students  
- Responsive and user-friendly web interface  

---

## System Architecture
- Frontend handles user interactions and dynamic UI updates  
- Backend exposes RESTful APIs for healthcare workflows  
- Database manages persistent storage of user and medical data  
- Asynchronous requests handled using AJAX for seamless UX  

---

## Impact and Outcomes
- Reduced manual coordination and administrative overhead  
- Improved response efficiency during medical emergencies  
- Centralized and reliable access to medical records  
- Enhanced overall user experience for on-campus healthcare services  

---

## Tech Stack
- Frontend: JavaScript, React.js, HTML, CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Communication: AJAX
